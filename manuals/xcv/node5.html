Manipulating a stack of images

To try the following functions, load another image using File so there are now two images on the stack.

1.
To scroll through the deck, use the 'PageUp' and 'PageDown' keys to move forwards and backwards through the deck respectively.

2.
Use '+' to add a column of panes to the stack manager and '-' to remove a column. Type '-' to return to the original display. The window can be resized to better fit the images by dragging the window corner with the mouse.

3.
Use CTRL and '+' to add a row of panes to the stack manager and CTRL and '-' to remove a row. Type CTRL and '-' to return to the original display.

4.
When several images are displayed, clicking on an image makes that image the active pane.

5.
To remove the top image, use the menu item File .
