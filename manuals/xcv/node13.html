Viewing epipolar geometry

To view the epipolar geometry between images either load or compute an F matrix for the two views.

  • To load an F matrix select Multiview . If the two images loaded are called basement00.mit and basement01.mit then the standard name for the F matrix would be basement.00.01.F.

  • To compute an F matrix first compute or load the corners from both images. Then select Multiview .

Now pressing the left mouse button in one of the windows displays the epipolar line of the selected point in the other image. Pressing the middle mouse button will leave the point and corresponding line on the display. To stop the display of epipolar lines select Multiview .
