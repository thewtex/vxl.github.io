Printing

As well as allowing the user to load and remove images, the File menu also has selections to print the image.

  • Selecting File prints the active image to a Postscript file. Entering a value larger than 1 as the 'Reduction factor' will reduce the resolution of the image saved by that factor, i.e. choosing a reduction factor of 4 will save an image with only 1/4 the detail of the original. There is also an option whether or not to save any spatial objects (circles, lines, points) drawn on the image.
