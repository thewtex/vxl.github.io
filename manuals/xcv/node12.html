Two Views

Start xcv with two images. If the image file names are basement00.mit and basement01.mit for example then start xcv by typing:

xcv basement0[01].mit
or,
xcv basement00.mit basement01.mit

To compute F or H matrices we will first need to load or compute corners in both images.

  • Use Multiview to load a file containing corners. The corners for the image basement00.mit will traditionally be contained in the file basement00.corners.

  • Use Multiview to compute the corners.

Note that if you start xcv with more than two images loaded then when performing a two view relation you will be asked to select which two images the operation is to be performed on. Do this by clicking the left mouse button once on each required image.
