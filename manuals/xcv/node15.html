Viewing planar transformations

To view a planar tranformation between images either load or compute an H matrix.

  • To load an H matrix select Multiview . A file containing an H matrix normally ends in a .H suffix, for example
    basement.00.01.groundplane.H.

  • To compute an H matrix, select Multiview .

Now pressing the left mouse button in one of the images displays the selected point as a cross on the other image. To hid this display select Multiview . Pressing the middle mouse button will leave the point and corresponding cross on the display.
