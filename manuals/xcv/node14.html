Corner matches and tracks

Computing a fundamental matrix also computes correspondences between the corners in each image.

  • Choosing Multiview will display a track on each matched point showing its corresponding position in the other image. Note that displaying the tracks will make it difficult to select a point and you may have to remove them from the display (by clicking the right mouse button on the image and choosing Line Tracks .) to select a point.

Corner matches can also be loaded using Multiview . The file containing corner matches between basement00.mit and basement01.mit would normally be called basement.00.01.cc.
