Adaptors provided with VGUI

For some of the more commonly used GUI toolkits VGUI provides an easy way to create menus, dialogue boxes and add tableaux. Using this VGUI pattern to create your applications also means that they are easily ported from one GUI toolkit to another. Thus it is possible to compile the same application to run with a different GUI on different platforms.

GUI toolkits that are supported or are intended to be supported in the near future are:

  • GTK (GIMP Toolkit):

    Information at http://www.gtk.org/

  • FLTK (Fast Light Toolkit) Information at http://fltk.easysw.com/

  • Motif

  • Qt

    Information at http://www.troll.no/developer/

  • GLUT (OpenGL Utility Toolkit)

    Information at http://www.opengl.org/Documentation/GLUT.html

  • glX

  • MFC (Microsoft Foundation Classes)
