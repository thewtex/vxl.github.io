VXL 1.0 beta release


The gzipped tarfile vxl-1.0-beta.tgz (3Mb) contains the official distribution from the Leuven repository.

Because it is a release distribution, it will not change in response to bug fixes or design changes.
If fact it will not change at all. It is frozen.
