Downloading CVS


To use CVS, you must have it installed on your system. You may wish to use the Cygwin tools on Windows platforms, or WinCVS which provides a very nice GUI to CVS. CVS (Concurrent Versions System) is a version control system. If you want to know more about it there is documentation at: http://www.loria.fr/~molli/cvs/doc/cvs_toc.html
